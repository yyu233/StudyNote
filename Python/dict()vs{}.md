## Performance ##
``` {} ``` is faster than ```dict()```. Use ```timeit``` module to measure the speed. But the difference is minor. 

## Functionality ##
 ``` dict() ``` accepts generator and as such.     
 ``` {} ``` can be useful when applying dictionary comprehension
